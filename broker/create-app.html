📦 Create the boxlite GitHub App

One click creates the App with every permission + setting already correct (GitHub's App Manifest flow). You just authenticate and confirm — nothing here can act without you.

Owner: the button targets the boxlite-ai org. To create it on your personal account, change the form action to https://github.com/settings/apps/new.

If GitHub says the name boxlite is taken (App names are globally unique), edit "name" in the script below to boxlite-reviewer and reload.

After you click “Create GitHub App”

  1. GitHub redirects you to https://boxlite.ai/app-created?code=XXXX — copy the code value from the address bar.
  2. Deploy the broker in one command (needs gh auth + npx wrangler login first): 
    cd broker
bash deploy.sh XXXX
    It exchanges the code for the App ID + key, loads them as Worker secrets, deploys, and prints your Worker URL + the final one-field step.

Full details in broker/SETUP.md.
